What is Frontend Development

It's everthing we see, click and interact with on a website.
Involves building the user interface(UI) jof web application.

Why it matters?

  • It shapes the first impression of a website.
  • Affects usability, accessibility and user experience (UX).
  • A Smoth, Responsive interface = Happy users.

Real-life anology

Think of it like a car inter. What the driver interacts with

Frontend is the dashbord, botons and screen

Backend is the engine of the entire site

Core Tecnologies

HTML

Builds the structure (headings, paragraphs, fonts) .

CSS

Adds style (color, layout, fonts).

JS

Adds interactivity (clicks, animations, logic).
